# GUI 使用说明

运行 `python main.py` 后进入交互式界面。界面按工作流组织，避免把结构振动和麦克风声学处理混在一起。

## 1. 工作流选择

顶部工作流包含：

| 工作流 | 用途 |
|---|---|
| 副车架振动数据处理 | 常规结构振动数据、PSD 传递率、热力图、减振率 |
| 5.22 + 5.23 同名平均 | 左右激振位置同名工况平均 |
| 麦克风声学数据处理 | 三通道麦克风声压级分析 |

选择不同工作流后，界面只显示该工作流相关选项。

## 2. 数据选择

左侧选择 `Raw_data` 下的日期文件夹。程序默认不自动勾选任何数据，需要人工确认后勾选。

支持：

- 搜索文件名；
- 全选当前筛选结果；
- 清空；
- 反选。

数据名称默认使用 `.mat` 文件名。图形样式页可临时修改“图中显示名”，不会改动原始文件名。

## 3. 出图类型

副车架振动流程中，出图类型按用途分组：

- 基础曲线：单输出、四点平均、六点平均、输入 PSD、输入力 PSD、FRF；
- 对比评价：归一化热力图、总振级、平均减振率。

总振级、平均减振率、热力图的统计测点由数据格式表自动决定：

- 含左中/右中：六点；
- 不含左中/右中：四点。

麦克风流程中可选择：

- FFT 声压级；
- 总声压级；
- 倍频程声压级；
- 麦克风位置；
- 同名前缀平均。

麦克风流程提供独立的频率上下限输入。该频率范围同时控制 FFT 声压级显示、总声压级积分和分数倍频程频带筛选。分数倍频程支持 `1/3`、`1/4`、`1/6`、`1/10`、`1/20` 倍频程。

## 4. 图形样式

图形样式页只显示本次已选数据或平均后的组。

每行可设置：

- 图例顺序；
- 原始文件名；
- 图中显示名；
- 曲线颜色；
- 线宽；
- 亮度。

点击色块可选择颜色。线宽和亮度使用下拉选项。

## 5. 保存与统计

未勾选“保存本次生成图像”时，只弹出图像窗口。

勾选后保存到指定输出目录，默认：

```text
output/figures/
```

可设置：

- 图片格式；
- DPI；
- 是否导出 Excel；
- 频率范围；
- 分析时长；
- 处理模式。

## 6. 运行检查

运行前建议确认：

1. 数据文件夹是否正确。
2. 输入基准是否使用自动规则或已手动指定。
3. 频率范围是否符合当前汇报需求。
4. 出图类型是否只勾选需要的内容。
5. 是否需要保存本次生成图像。
6. 图中显示名和颜色是否已经调整。
