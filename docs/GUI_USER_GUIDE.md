# AITO 数据处理程序界面使用说明

## 1. 选择工作流

启动后先选择工作流：

- `副车架振动数据处理`：处理结构振动传递率、PSD、FRF、热力图、总振级和平均减振率。
- `5.22 + 5.23 同名平均`：只处理两个日期下同名 `.mat` 文件的平均结果。
- `麦克风声学数据处理`：处理 5.31 三通道麦克风数据。
- `6.11 实车声振数据处理`：处理实车道路试验中的麦克风、半轴三向加速度和副车架单向加速度数据。

## 2. 选择数据

数据表中默认不勾选任何文件。用户可搜索、全选当前显示、清空或反选。表格会显示原始文件名、图中显示名、日期、数据结构、输入基准和状态。

## 3. 选择图型

每个工作流只显示相关图型。副车架流程分为基础曲线和评价图；其中“加速度 FFT（1 Hz）”会按测点分别出图，同一测点叠加所选工况，不同测点不会画在同一张图中。麦克风流程只显示 FFT 声压级、总声压级和分数倍频程声压级；6.11 实车流程额外提供路谱归一化声压级、声振相干性和声振关联散点图。

## 4. 设置参数

常用参数包括频率范围、输入基准、处理模式、保存格式、DPI 和保存位置。输入基准可选择自动识别，也可手动指定 `Data第1行（力传感器）` 或 `Data第2行（左输入）`。麦克风流程单独提供 FFT 频段、总声压级/倍频程频段和倍频程类型。

## 5. 图形样式

样式页只列出本次实际参与出图的曲线或平均组。可修改图中显示名、颜色、线宽、线形、亮度和顺序。颜色通过点击色块选择。

标题设置支持两种方式：

- `完整标题覆盖`：填写后直接替换每张图的标题；
- `标题前缀/标题后缀`：完整标题为空时，在默认标题前后追加文字。

线形可选实线、虚线、点划线、点线；也可以在每条曲线行单独覆盖。

## 6. 确认运行

确认页会汇总本次任务。若缺少 no-load、DVA、同名文件、麦克风通道不足或 6.11 实车通道数量不符合数据格式表，程序会在运行前或运行中给出明确提示。
