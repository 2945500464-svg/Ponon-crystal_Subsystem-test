# 数据结构说明

本项目的 `.mat` 文件统一要求包含：

```text
shdf.Absc1Data
shdf.Data
```

其中：

- `Absc1Data`：时间数据。
- `Data`：通道数据，通常为 `M x N`，`M` 为通道数，`N` 为采样点数。

程序按固定行号解释通道。不同试验日期批次的数据结构不同，不能混用。

## 1. 5.8 批次数据结构

5.8 数据为早期结构，没有插入激振器加速度通道。

| Data 行号 | Python 索引 | 通道含义 | 常用用途 |
|---:|---:|---|---|
| 1 | 0 | 左输入 / 输入加速度基准 | 5.8 输入归一化基准 |
| 2 | 1 | 右输入 | 备用输入 |
| 3 | 2 | 输入力 | 输入力 PSD / FRF 可选 |
| 4 | 3 | 左前 | 四点平均输出 |
| 5 | 4 | 左后 | 四点平均输出 |
| 6 | 5 | 左动力吸振器 / 历史中间测点 | 可选输出 |
| 7 | 6 | 右前 | 四点平均输出 |
| 8 | 7 | 右后 | 四点平均输出 |
| 9 | 8 | 右动力吸振器 / 历史中间测点 | 可选输出 |

5.8 四点平均默认使用：

```text
左前、左后、右前、右后
Data 索引：[3, 4, 6, 7]
```

5.8 默认输入基准：

```text
Data 第1行，Python 索引 0
```

## 2. 5.10 / 5.11 / 5.12 批次数据结构

后续批次在力传感器下方插入了激振器加速度通道，因此输出测点整体后移。

| Data 行号 | Python 索引 | 通道含义 | 常用用途 |
|---:|---:|---|---|
| 1 | 0 | 左输入 | 输入基准可选 |
| 2 | 1 | 右输入 | 5.12 当前部分图使用的输入基准 |
| 3 | 2 | 输入力 | 输入力 PSD / FRF 可选 |
| 4 | 3 | 激振器加速度 | 输入基准可选 |
| 5 | 4 | 左前 | 六点平均输出 |
| 6 | 5 | 左后 | 六点平均输出 |
| 7 | 6 | 左中 | 六点平均输出 |
| 8 | 7 | 右前 | 六点平均输出 |
| 9 | 8 | 右后 | 六点平均输出 |
| 10 | 9 | 右中 | 六点平均输出 |

5.11 / 5.12 六点平均默认使用：

```text
左前、左后、左中、右前、右后、右中
Data 索引：[4, 5, 6, 7, 8, 9]
```

5.12 在当前归一化指标脚本中默认输入基准：

```text
Data 第2行，Python 索引 1
```
