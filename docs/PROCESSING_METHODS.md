# 数据处理与出图方法

## 1. PSD 计算

PSD 使用 Welch 方法计算：

```text
S(f) = Welch(x(t))
```

项目默认参数：

```text
fs = 48000 Hz
target_df = 1/20 Hz
window = Hann
scaling = density
```

程序会根据目标频率分辨率和实际数据长度自动确定 `nfft`，避免硬编码采样点数。

## 2. 输入归一化 PSD 传递率

对第 `j` 个输出测点：

```text
R_j(f) = S_yy,j(f) / S_xx(f)
```

其中：

- `S_xx(f)`：输入基准通道 PSD。
- `S_yy,j(f)`：第 `j` 个输出测点 PSD。

该指标消除了不同工况输入能量不完全一致造成的影响，更适合比较不同方案本身的传递能力。

## 3. 多测点平均

多测点平均必须在 PSD 比值的线性域进行：

```text
R_avg(f) = sum(w_j * R_j(f)) / sum(w_j)
```

不应先将 `R_j(f)` 转为 dB 后再平均。

常用测点：

- 5.8：左前、左后、右前、右后四点等权平均。
- 5.11 / 5.12：左前、左后、左中、右前、右后、右中六点等权平均。

## 4. 频段均值

按当前项目主评价公式：

```text
L_B = -10 * log10( (1 / N_B) * sum_{f_i in B} R_avg(f_i) )
```

其中：

- `B`：目标频段。
- `N_B`：频段内频率点数量。
- `L_B`：频段均值的 dB 表达。

解释：

- `L_B` 越大，代表该频段内输入归一化 PSD 传递率越低。
- 目标频段可取 `80-100 Hz`、`90-110 Hz`、`70-140 Hz`。

## 5. 相对 no-load 减振率

先计算相对 no-load 的频段差值：

```text
Delta L_B = L_B,scheme - L_B,no-load
```

再计算减振率：

```text
DR_B = [1 - 10^(-Delta L_B / 10)] * 100%
```

解释：

- `DR_B > 0`：相对 no-load，频段平均传递率降低。
- `DR_B < 0`：相对 no-load，频段平均传递率放大。

## 6. 总振级与 PSD 传递率的区别

总振级是绝对输出 PSD 的频带积分，例如：

```text
L_a = 10 * log10( integral(PSD_out(f), f in B) / a0^2 )
```

输入归一化 PSD 传递率是：

```text
R_j(f) = PSD_out,j(f) / PSD_in(f)
```

两者不能混用：

- 如果要评价输出绝对能量，用总振级。
- 如果要评价方案对结构传递能力的影响，用输入归一化 PSD 传递率。

当不同工况输入 PSD 差异明显时，应优先使用输入归一化 PSD 传递率。
