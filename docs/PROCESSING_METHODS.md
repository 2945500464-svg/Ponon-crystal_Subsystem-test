# 数据处理方法

## 副车架结构振动

程序从 `.mat` 文件中读取 `shdf.Absc1Data` 和 `shdf.Data`。不同日期的数据结构由 `Raw_data/数据格式.xlsx` 或内置规则确定。

输入归一化 PSD 比值：

```text
R_i(f) = S_yy,i(f) / S_xx(f)
```

PSD 传递率：

```text
TL_i(f) = 10 lg[R_i(f)]
```

多测点等权平均先在线性域进行：

```text
R_avg(f) = Σ w_i R_i(f) / Σ w_i
```

平均减振率：

```text
L_band = 10 lg[mean(R_avg(f))]
ΔTL = L_scheme - L_no-load
DR = [1 - 10^(ΔTL/10)] × 100%
```

热力图、总振级和平均减振率的统计测点自动判断：包含左中、右中时使用六点等权平均；否则使用左前、左后、右前、右后四点等权平均。

## 5.22 + 5.23 同名平均

两侧同名工况先分别完成传递率计算，再按线性域结果平均，最后转换为 dB 出图。该流程用于削弱左右激励位置差异带来的偶然性。

## 麦克风声学

麦克风数据单位为 Pa，参考声压为：

```text
p0 = 20e-6 Pa
```

FFT 声压级图采用 1 Hz 频带积分：

```text
Lp,1Hz(f) = 10 lg[Σ PSD(f) Δf / p0^2]
```

总声压级在指定频段内积分：

```text
Lp,total = 10 lg[Σ PSD(f) Δf / p0^2]
```

分数倍频程声压级按所选 `1/N` 倍频程中心频率和上下限频带积分，当前支持 `1/3`、`1/6`、`1/12`、`1/24`。

图形样式只改变显示效果，不改变上述计算公式。标题、颜色、线宽、线形和亮度调整不会影响数据结果。
