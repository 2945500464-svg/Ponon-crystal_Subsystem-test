# 数据处理方法

本文档说明程序中的主要计算口径。符号中的 `x` 表示输入基准通道，`y_j` 表示第 `j` 个输出测点。

## 1. 采样与频段

默认采样频率：

```text
fs = 48000 Hz
```

默认结构振动观察频段：

```text
70-140 Hz
```

程序支持在 GUI 中修改起止频率、分析时长、频率分辨率和处理模式。

## 2. 分段处理

支持两种处理模式：

| GUI 名称 | 内部模式 | 含义 |
|---|---|---|
| 单段分析 | `single_segment` | 从指定起点截取一段数据分析；长度不足时允许零填充 |
| 分段平均 | `segment_average` | 将长时程按指定时长分段，每段计算后在线性域平均 |

所有通道在频域分析前去均值并乘 Hann 窗。

## 3. PSD 传递率

输入通道 PSD：

```text
S_xx(f)
```

输出测点 PSD：

```text
S_yy,j(f)
```

输入归一化 PSD 比值：

```text
R_j(f) = S_yy,j(f) / S_xx(f)
```

dB 表达：

```text
L_j(f) = 10 lg[R_j(f)]
```

该指标表示输出测点相对输入的频域传递能力。若不同工况输入能量不一致，应优先比较 `R_j(f)` 或 `L_j(f)`，不要直接比较输出 PSD。

## 4. 多测点平均

多测点平均必须在线性域进行：

```text
R_avg(f) = sum(w_j R_j(f)) / sum(w_j)
```

等权平均时：

```text
w_j = 1
```

平均后的 dB 曲线：

```text
L_avg(f) = 10 lg[R_avg(f)]
```

程序中四点或六点平均图均按此口径计算。

## 5. 热力图

相对基准工况的频率-方案改善量：

```text
I_R(f) = 10 lg[R_avg,scheme(f) / R_avg,baseline(f)]
```

其中 `baseline` 可以是 `no-load` 或 `DVA`。

解释：

- 负值：相对基准输入归一化 PSD 比值降低，通常表示传递能力降低。
- 正值：相对基准输入归一化 PSD 比值升高，可能表示该频点传递增强。

热力图用于判断方案是否形成连续频段改善，而不只是在某个窄频点削峰。

## 6. 频段总振级

程序中的总振级图基于输入归一化 PSD 传递率，不是绝对加速度声振级。频段内先对 `R_avg(f)` 做均值或积分口径统计，再转为 dB：

```text
L_B = 10 lg[mean(R_avg(f_i))]
```

`f_i` 为频段 `B` 内的频率点。

## 7. 平均减振率

以 `no-load` 为基准：

```text
Delta L = L_scheme - L_no-load
```

减振率：

```text
DR = [1 - 10^(Delta L / 10)] × 100%
```

若 `DR > 0`，表示相对 no-load 降低；若 `DR < 0`，表示相对 no-load 放大。

## 8. 5.22 + 5.23 同名平均

该流程用于同名工况的重复试验综合。程序先分别计算 `5.22` 和 `5.23` 的输入归一化 PSD 比值，再对同名工况在线性域平均：

```text
R_avg_pair(f) = [R_5.22(f) + R_5.23(f)] / 2
```

平均后再转为 dB 出图。

## 9. 力-加速度 FRF

当数据中存在力传感器和激振器加速度通道时，可计算输入点响应比：

```text
H(f) = S_aa(f) / S_ff(f)
```

dB 表达：

```text
H_dB(f) = 10 lg[H(f)]
```

该图用于检查激振输入点响应与力输入之间的频域关系。

## 10. 麦克风声学处理

麦克风原始单位为 Pa，参考声压：

```text
p0 = 20 × 10^-6 Pa
```

声压级：

```text
L_p = 10 lg[p_rms^2 / p0^2]
```

FFT 声压级图按 GUI 中填写的麦克风频率范围显示各工况的频率点声压级，默认范围为 70-140 Hz。

总声压级在目标频段内累加压力均方值：

```text
L_total = 10 lg[sum(p_rms(f_i)^2) / p0^2]
```

倍频程图按 GUI 中填写的麦克风频率范围筛选有交集的频带，支持：

- 1/3 倍频程；
- 1/4 倍频程；
- 1/6 倍频程；
- 1/10 倍频程；
- 1/20 倍频程。

若勾选同名前缀平均，例如 `plan1_1.mat` 和 `plan1_2.mat`，程序会先按前缀 `plan1` 分组并在线性声压能量域平均，再输出三类声学图。
